# welcome to my shopping cart demo

# usage

## run the demo
make sure you have configured $GOPATH, go to the directory $GOPATH/src, 
and run following commands to run the demo

```

go get "github.com/wy3148/shopping_cart"

cd github.com/wy3148/shopping_cart
go run main.go
```
